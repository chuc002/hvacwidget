How to Embed the Widget

There are multiple ways to embed the widget into your website:

Add this HTML code to your website:

<iframe 
  src="https://hvac-subscription-widget.example.com/widget" 
  width="100%" 
  height="800px" 
  frameborder="0">
</iframe>

This is the simplest way to embed the widget. Just replace the src URL with your actual widget URL.

Add this JavaScript code to your website:

<div id="hvac-widget-container"></div>
<script src="https://hvac-subscription-widget.example.com/js/embed.js"></script>
<script>
  HVACWidget.init({
    container: 'hvac-widget-container',
    companyName: 'Your HVAC Company',
    primaryColor: '#0F766E'
  });
</script>

This method allows for more customization options like changing colors and company name.

For WordPress sites, install our plugin and use this shortcode:

[hvac_subscription_widget company="Your HVAC Company" primary_color="#0F766E"]

You can add this shortcode to any page or post in your WordPress website.

Benefits of Embedding

  • Allow customers to subscribe to maintenance plans directly from your website
  • Increase revenue with automated subscription processing
  • Reduce administrative work with digital payment handling
  • Seamlessly integrate with your existing CRM via Zapier
